# corii.xyz

My personal website, built with Astro and Tailwind CSS.


### Prerequisites

To build this project, you need:
- [node.js](https://nodejs.org/en/download)

### Build + Run

Install dependencies: 

    npm i

Run:

    npm run dev

## Built With

  - [Astro](https://astro.build/)
  - [Tailwind CSS](https://tailwindcss.com/)
